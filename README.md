# Quiz Console App (Java)

A console-based Quiz Application built using Java.  
This project demonstrates Object-Oriented Programming (OOPS) concepts and clean layered architecture.

---

## Features

- Multiple Choice Questions (MCQ)
- Array of Objects Implementation
- Constructor-Based Object Initialization
- Encapsulation (Getters/Setters)
- Layered Structure (Main → Service → Model)
- Clean Console Output

---

## Concepts Used

- Classes & Objects
- Constructors
- Arrays
- Encapsulation
- Method Overriding (toString)
- For-Each Loop

---

## How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/quiz-console-app.git
```

2. Compile:

```
javac *.java
```

3. Run:

```
java Main
```
